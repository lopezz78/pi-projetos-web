# Inclui+

## Descrição
Aplicativo desenvolvido para Compra, Venda e Doações de produtos, focado em pessoas com necessidades especiais.

## Tecnologias Utilizadas
- HTML5, CSS3, JavaScript
- Node.js
- Banco de Dados: MySQL / XAMPP / PHP / Apache / InfiniftFree

## Estrutura do Projeto
Explicação da estrutura de pastas e arquivos do projeto:
- api/ - Contém scripts de backend em PHP para manipulação do banco de dados.
- Inclui+ - Pasta com as Fotos utilizadas no Projeto.
- .js - Contém arquivos JavaScript para funcionalidades do frontend.
- .css - Contém arquivos CSS para estilização do frontend.
- .html - Contém arquivos HTML para as páginas do site.
